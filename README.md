## 🚀 Quick Start (Web)

### Option A — Run in Google IDX (Project IDX / Firebase Studio)
1. Open your project folder in IDX
2. The `.jsx` file is a **React component** — you need a React scaffold around it
3. Follow `ANDROID_GUIDE.md` to set up the full project

### Option B — Run locally
```bash
npx create-react-app ecb-tracker
cd ecb-tracker
# Replace src/App.js content with ecb-tracker.jsx content
npm start
```

## 🧠 Tech Stack

| Layer | Technology |
|-------|-----------|
| UI Framework | React 18 (JSX) |
| Styling | Pure CSS-in-JS (no Tailwind, no external UI lib) |
| Storage | `window.storage` API (Claude artifact persistent storage) |
| Fonts | Google Fonts — Outfit + DM Mono |
| State | React `useState`, `useEffect`, `useRef` |
| Build target | Web → Android via Capacitor |

---

## ⚠️ Important Notes

- The current `ecb-tracker.jsx` was built inside **Claude.ai Artifacts** which uses a special `window.storage` API for persistence
- When moving to a real React app, you must **replace `window.storage` with `localStorage`** (see `ANDROID_GUIDE.md`)
- All data is stored **locally on device** — no backend, no server, no cloud sync yet
